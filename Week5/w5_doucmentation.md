# Week 5 Documentation

## 1. Problem Statement

Create a Python program that calculates a customer's cafe bill automatically.

## 2. Inputs

- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 3. Outputs

- Receipt
- Total bill

## 4. Process Flow

1. Input customer name
2. Input quantity of coffee
3. Input quantity of tea
4. Input quantity of sandwich
5. Calculate total price
6. Print receipt

## 5. Constraints

- Quantities cannot be negative.
- Prices are fixed.
